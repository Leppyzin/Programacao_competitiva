## O que é um Algoritmo?

Um algoritmo é um conjunto de passos que devem ser executados de maneira lógica e bem estruturada, especificando passo a passo uma solução para um problema, sendo fundamental para o desenvolvimento de programas. Além disso, ele é sempre finito.

Veja a seguir um problema de exemplo e suas variações na forma de escrever os mesmos:

## Problema inicial

 O seu primeiro programa em qualquer linguagem de programação normalmente é o "Hello World!". 
Neste primeiro problema tudo o que você precisa fazer é imprimir esta mensagem na tela.



### Algoritmo Descritivo

```
1) Imprima na tela a seguinte mensagem **"Hello World!"**
```

### Pseudo-código

```
Imprima na tela "Hello World!"
```

### Algoritmo computacional


```
int main(){

    printf("Hello World!\n");

    return 0;
}
```
