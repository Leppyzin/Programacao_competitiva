# Expressões Matemáticas

## Atribuição

  Atribui o valor 5 à variável inteira ```a```:

  ```
#include <stdio.h>

int main(){
    int a;

    a = 5;

    return 0;
}
```

Atribui o valor que está na variável inteira ```b``` à variável inteira ```a```:

```
#include <stdio.h>

int main(){
    int a,b;

    a = 3;
    b = 9;

    a = b;

    printf("Veja o valor de: %d\n");

    return 0;
}
```

## Operadores Matemáticos

### Passaremos resumidamente por algumas operações que podemos fazer com as variáveis

Em breve, conseguiremos usar mais operações. Mas, por enquanto não é trivial sabermos disso. Temos que saber algumas operações básicas para depois avançarmos para algumas variáveis mais completas.

Em resumo, as operações matemáticas são descritas pela seguinte tabela:

|Operação|operador em Linguagem C|Exemplo|
|--------|-----------------------|-------|
|soma| + | s = a + b;|
|subtração| - |s = a - b;|
|Multiplicação| * |s = a * b;|
|Divisão| / |s = a / b;|
|Resto de divisão| % | s = a % b;|

em que ```s```,```a```,```b``` são variáveis numéricas, isto é, inteiras ou reais. Idealmente, tente sempre fazer as operações em variáveis de mesmo tipo.
